使用 GitHub Pages + Hexo 写博客

本篇文章简单记录一下借助 GitHub 搭建博客的流程。

1. 新建 GitHub Pages 站点

  1. 在GitHub上新建一个名为 username.github.io 的公开仓库，username 为当前 GitHub 账户的用户名；
  2. 进入仓库【Settings】-【Pages】选项，启用 GitHub Pages ，访问 http(s)://username.github.io 即可打开默认 Hello World 页面。

2. 本地部署 Hexo 博客框架

2.1 安装前提

  • Git

2.4 博客目录结构

  • _config.yml：网站的配置信息，可以在该文件中修改大部分的配置；
  • source：资源文件夹是存放用户资源的地方。除 _posts 文件夹之外，开头命名为 _ (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。
  • themes：主题文件夹。Hexo 会根据主题来生成静态页面。
